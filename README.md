# Osteopatía y Naturopatía – Web Profesional

Web profesional optimizada para SEO y rendimiento, desarrollada con React, TypeScript y Vite. Incluye optimización de imágenes, sitemap, indexación en Google y despliegue en producción.

Web en producción: https://osteopatiaynaturopatia.com

- SEO técnico avanzado
- Optimización de rendimiento
- Arquitectura limpia y mantenible
- Buenas prácticas de despliegue
- Experiencia de usuario responsive

---

## Objetivo del Proyecto

Desarrollar una web corporativa moderna, rápida y optimizada para buscadores que:

- Posicione en búsquedas locales
- Transmita profesionalidad y confianza
- Tenga excelente rendimiento (Core Web Vitals)
- Sea fácilmente mantenible
- Esté correctamente indexada en Google

---

## Stack Tecnológico

### Frontend:

- React
- TypeScript
- Tailwind CSS
- Shadcn/UI
- Framer Motion
- Wouter (routing ligero)
- Lucide React (iconos)
- Vite (build tool)

### Infraestructura y despliegue:

- Vercel (hosting y CDN)
- Dominio personalizado con HTTPS

---

## Arquitectura

- SPA desarrollada con React
- Routing semántico mediante Wouter
- Estructura modular por páginas y componentes
- Build optimizada con Vite
- Versionado automático de assets mediante hashing

El proyecto incluye estructura backend, pero la versión desplegada en producción utiliza únicamente el frontend.

## Optimización de Rendimiento

Se han aplicado distintas mejoras orientadas a reducir tiempos de carga y mejorar la experiencia del usuario:

- Optimización manual de imágenes
- Conversión a formato WebP
- Redimensionado y compresión de recursos gráficos
- Uso de `loading="lazy"` en imágenes
- Minificación y optimización automática en el proceso de build
- Generación de archivos con hash para evitar problemas de caché

El despliegue en Vercel permite servir los archivos a través de una CDN global.
